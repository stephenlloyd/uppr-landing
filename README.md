# uppr Landing Page

Static marketing site for [uppr.fit](https://uppr.fit).

## Deploy

This is a single `index.html` file. Deploy to Netlify, Cloudflare Pages, or any static host.

### Netlify
1. Connect this repo in Netlify dashboard
2. Build command: (none)
3. Publish directory: `.`
4. Custom domain: `uppr.fit`

### DNS (Namecheap)
Point `uppr.fit` to Netlify per their custom domain instructions.

## Structure

```
index.html          — the entire landing page
favicon.svg         — SVG favicon (copy from gymapp/public/)
apple-touch-icon.png — 180×180 (generate from tools/generate-icon.html)
```
